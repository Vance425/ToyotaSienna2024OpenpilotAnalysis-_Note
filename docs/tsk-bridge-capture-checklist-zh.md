# TSK Bridge 錄制超短清單

## 出門前

- 啟動 logger
- 確認 `IGN_ON` 後有持續錄
- 如果方便，準備打 marker

## 這次要錄什麼

目標不是純市區 stop-go。

目標是：

- 城市起步
- `ACC_ON`
- `LKAS_ON`
- 上快速道路 / 高速
- 保持一段穩定 assisted follow
- 最好再回到一般道路

## 最低要求

- 連續 `IGN_ON`
- 至少 `5-10` 分鐘
- 最好 `8-15` 分鐘

## 最重要 marker

只要記大概就夠：

- `ACC_ON`
- `LKAS_ON`
- `FREEWAY_ENTRY`
- `FREEWAY_EXIT`
- `C3X_WARNING`
- `C3X_ERROR`
- 明顯踩煞車解除跟車

## 這次最想打到的東西

我們不是要再多一個普通 `Grade B`。

我們要的是一段比較像：

- 比 `171414_000` 更完整
- 但還沒到 `190101_000`

也就是：

- `seed-heavy partial ramp`
- 往 `joined lifecycle` 再前進一步

## 不用太在意的事

不用精確記每個紅燈。

不用精確記每次 stop-go 秒數。

現在比較重要的是：

- ADAS 什麼時候開
- 什麼時候上快速道路 / 高速
- 有沒有 warning / disengage

## 最短版

如果只記一句話：

> 錄一段連續 `IGN_ON`，城市出發後開 `ACC/LKAS`，上快速道路或高速，維持一段 assisted follow，並記住大概的 `ACC_ON / LKAS_ON / FREEWAY_ENTRY / FREEWAY_EXIT`。
