# 目前根因

## 已確認的主因

目前 `comma 3X` 無法在你的 `2024 Toyota Sienna` 正常使用，核心原因不是單純安裝失敗，而是：

- 車端目前沒有可用的 `TSK`
- `comma 3X` 也無法自動成功抓取 `Toyota Security Key`

因此結果是：

- 系統無法完成車端安全機制所需的條件
- `comma 3X` 雖然存在，但不能進入正常可用狀態

## 問題的本質

這不是一般的「接頭不對」問題，也不是一般的「車型列表沒加進去」就能解的問題。

目前真正的阻塞點是：

`如何透過 CAN log 分析，找出可取得 TSK 的方法`

## 專案目標已更新

本專案現在的主要目標是：

1. 收集可用的 CAN log
2. 從 log 中識別與 security / challenge-response / key material 相關的訊息模式
3. 判斷是否存在可重現的 TSK 取得途徑
4. 把結果整理成後續實作或研究輸入

## 不再是目前優先的事情

以下事情不是現在的第一優先：

- 一般安裝教學
- 單純換 mount 或重新走線
- 先追求道路測試

現在第一優先是把 `TSK 取得方法` 研究出來。
