# `0x116` Phase-Boundary Nibbles

## 目標

專看 `0x116 tail` 裡目前最值得追的三個 nibble：

- `n0`
- `n1`
- `n3`

要回答的問題是：

- 哪些 nibble 會在 phase 切換點出現固定轉折
- 哪些 nibble 只是持續高熵漂移

## 樣本 1: `184921`

序列摘錄：

- `24 00` -> `n0=1/5`, `n1=5/4`, `n3=d/6`
- `25 00` -> `n0=9/d/1/6`, `n1=d/4/5/6`, `n3=d/3/0/a`
- `26 00` -> `n0=a/e`, `n1=4/a`, `n3=a/b`
- `27 00` -> `n0=2/6/a/e`, `n1=3/f/6/8`, `n3=1/8/a/f`
- `28 00` -> `n0=2/6/a`, `n1=f/1/a`, `n3=d/d/9`
- `29 00` -> `n0=e/2`, `n1=f/b`, `n3=f/5`

### 判讀

`n0`

- 最像 phase 內 rolling nibble
- 在多個 phase 內都會走出 `2 -> 6 -> a -> e` 這種 `+4` 風格
- phase 切換時，常會重設到新的起點

`n1`

- 沒有 `n0` 那麼穩
- 但在某些 phase 帶內會出現局部重複或小範圍聚集
- 比較像過渡混合區，不像主 counter

`n3`

- 常在 phase 內快速跳動
- 沒看到固定步進
- 但在 phase 切換後，常會換到一組新的活動區間

## 樣本 2: `185451`

序列摘錄：

- `09 09` -> `n0=1`, `n1=d`, `n3=1`
- `09 0a` -> `n0=5/6/a/e/2/6/a`, `n1=6/6/1/2/e/c/b`, `n3=8/3/6/e/9/a/f`
- `0a 0c` -> `n0=e`, `n1=c`, `n3=1`
- `0d 0e` -> `n0=6/a/2/6/7/b`, `n1=9/5/7/7/9/7`, `n3=3/b/f/f/d/f`

### 判讀

`n0`

- 最有規律
- `09 0a` phase 中，幾乎一直表現為 `+4` 節奏
- 到 `0d 0e` 後，仍保留 `+4` 痕跡，只是混入 `+8` 與 `+1`

`n1`

- 在 `0d 0e` phase 中有明顯重複：`7` 多次出現
- 比 `n3` 更像有半穩定停頓

`n3`

- 在 `0d 0e` 中也有重複：`f` 多次出現
- 但整體仍比 `n1` 更跳

## 合併結論

### `n0`

目前最像：

- phase-aware rolling nibble

它不像獨立於 phase 的全域 counter，但也不像純高熵噪聲。

### `n1`

目前最像：

- phase 內的半穩定過渡位

它會停頓、重複，但不呈現乾淨步進。

### `n3`

目前最像：

- 更高動態的過渡位

有時會重複，但整體不如 `n1` 穩。

## 目前最實用的 tail 模型

現在可以把 `0x116 tail` 暫時拆成：

- `n0`: rolling candidate
- `n1`: semi-stable transition
- `n3`: dynamic transition
- `n4-n7`: auth-heavy region

## 對後續分析的意義

如果之後真的要逼近可重放或可預測結構，最值得先看的已經不是整個 tail，而是：

1. `n0` 是否可由 phase/state 推出下一步候選
2. `n1/n3` 是否在某些 family state 下進入停頓區

這比直接拿 32-bit tail 去猜公式更實際。
