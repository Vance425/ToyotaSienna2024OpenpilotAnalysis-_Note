# 實戰版總覽

## 目標

用最小風險的方式，判斷你的 `2024 Toyota Sienna` 是否適合導入 `comma 3X`。

## 關鍵現況

- 官方支援頁面可見 `Toyota Sienna 2018-2020`
- 官方文件把 `Toyota Sienna 2021+` 列在 `Toyota Security` 類別
- 社群資料顯示 `2021-2023 Sienna Hybrid` 有可用案例
- 社群資料對 `Early 2024 MY` 保留可能性，但資料不足

## 這代表什麼

- 你的車不能當成官方支援 Toyota 直接裝
- 你的主要風險不是 mount，而是 `TSK / SecOC`
- 是否值得投入，取決於 `VIN / 製造年月 / 市場版本 / 韌體`

## 成功條件

- 你完成車輛資料盤點
- 你接受非官方支援事實
- 你接受先做靜態研究
- 你能判定自己的車不是明顯紅燈條件
