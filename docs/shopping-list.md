# 採購清單

## 最低必要

- [ ] comma 3X
- [ ] 相容 mount
- [ ] OBD-C 線材
- [ ] Toyota 對應線束

## 重要說明

社群資料把 `2021-2023 Sienna Hybrid` 對應到 `Toyota Harness A`，但這不能直接當成 `2024` 已驗證結論。

對 `2024` 的正確作法是：

- 先把 `Harness A` 當成優先假設
- 再交叉確認實車接口與社群樣本
- 不要在未確認前一次買滿所有配件

## 不建議先買的

- [ ] 額外備用線束
- [ ] 多組 mount
- [ ] 非必要轉接件
- [ ] 高風險專用測試配件
